# ssbh_lib
A parsing library for SSBH formats in Rust. Parsing is implemented for the following types:  
* Hlpb (`.nuhlpb`)
* Matl (`.numatb`)
* Modl (`.numdlb`)
* Mesh (`.numatb`)
* Skel (`.nusktb`)
* Anim (`.nuanmb`)

# Credits
The original C# file formats and parsing code can be found in the [SSBHLib](https://github.com/Ploaj/SSBHLib) repo.
